# OpenStack DevStack Lab

## Objective

Deployment of an OpenStack environment using DevStack in a controlled laboratory setup.

## Environment

- OS: Ubuntu 22.04 LTS
- Deployment: DevStack
- Architecture: Single-node

## Goals

- Understand OpenStack services
- Configure networking (Neutron)
- Deploy instances (Nova)
- Access Horizon dashboard
